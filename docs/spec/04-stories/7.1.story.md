# Story 7.1: 메트릭 이상 탐지 (z-score 기준선)

Status: done

## Story
**운영자**는 메트릭이 평소 패턴(통계 기준선)에서 벗어나면 이상 알람을 받는다. *(목적: 고정 임계치를 매번 손으로 튜닝하지 않고도 평소와 다른 거동을 인지하기 위해)*

## Acceptance Criteria
1. Anomaly 타입 rule은 각 시계열을 이동평균 ± k·σ 기준선 대비 z-score(이상 점수)로 변환한다 (FR-CF7.1)
2. 이상 점수는 기존 임계 파이프라인에 통과되며 목표치는 σ 단위 밴드 폭(k·σ)이다
3. 기준선은 평가 윈도우 직전 데이터포인트들로 형성되고 최신 데이터포인트가 그에 대해 평가된다
4. 최신 데이터포인트가 밴드 내면 이상 알람(매칭 샘플)은 생성되지 않는다
5. 생성된 이상 알람은 일반 알람과 동일 하류(PII·DLQ·감사·핸드오프)로 흐른다
